## üß† 2026 SEO Principles We're Following

Based on current research:

1. **Brand Signals > Keywords**
   - Branded search volume (people searching "made right studio")
   - Consistent NAP (Name, Address, Phone) across web
   - Social media mentions and engagement

2. **Entity Clarity**
   - Clear structured data telling Google you're a web design studio
   - Geographic signals (Columbia, SC) in all content
   - Service type consistency across platforms

3. **Topical Depth**
   - Multiple pages about web design topics
   - Blog posts demonstrating expertise
   - Portfolio showing actual work

4. **User Signals**
   - Low bounce rate (engaging content)
   - Time on site (compelling portfolio)
   - Return visitors (valuable content)

5. **AI Engine Optimization**
   - Content structured for ChatGPT, Claude, Perplexity
   - Clear, semantic HTML
   - Natural language, not keyword stuffing
